Steel Profiles REST API


A simple REST API for requesting data for common European steel profiles.

Check out the Docs for more information.

Endpoints

  • GET /api/{profile_type}
    E.g. to get data for all HEA profiles, visit GET /api/HEA
  • GET /api/{profile_type}/{dimension}
    E.g. to get data for the profile HEA120, visit GET /api/HEA/120

Units

The metric units used for the steel dimensions in this API are:

  • h: mm
  • b: mm
  • tw: mm
  • tf: mm
  • r: mm
  • d: mm
  • A: cm²
  • G: kg/m
  • Iy: cm⁴
  • Wy: cm³
  • iiy: cm
  • Iz: cm⁴
  • Wz: cm³
  • iiz: cm

Supported profiles

For now, the supported profiles are HEA, HEB, HEM, IPE, IPN and UPN .

References

For creating this API, data from prontubeam were used.
